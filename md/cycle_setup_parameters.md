# `cycle_setup_parameters.json` 文件说明

本文档旨在详细解释 `cycle_setup_parameters.json` 文件中各个参数的意义和作用。该文件是超临界CO2布雷顿循环与有机朗肯循环（SCBC-ORC）联合循环模拟器 (`full_cycle_simulator.py`) [1][2][5] 及参数优化脚本 (如 `genetic_algorithm_optimizer.py` [4]) 的核心配置文件。它存储了运行一次完整循环模拟所需的所有关键参数 [5]。

## 一、文件结构

`cycle_setup_parameters.json` 文件采用 JSON (JavaScript Object Notation) 格式，这是一种轻量级的数据交换格式，易于人阅读和编写，也易于机器解析和生成。

文件通常包含以下几个主要部分（或称"键"），每个部分下有更详细的参数设置：

1.  `common_parameters`: 通用参数，如环境条件。
2.  `scbc_parameters`: SCBC (超临界二氧化碳布雷顿循环) 相关参数。
3.  `orc_parameters`: ORC (有机朗肯循环) 相关参数。
4.  `heat_exchangers_common` (或在各循环参数内定义): 换热器通用参数，或特定于SCBC/ORC的换热器参数。

## 二、关键变量与参数计算

系统基于四个关键变量进行参数计算：

1. **SCBC透平入口温度 (T5)**：
   - 参数名：`T5_turbine_inlet_C`
   - 默认值：599.85°C
   - 作用：决定循环的最高温度，影响热效率和火用效率

2. **SCBC主循环压比 (PR)**：
   - 参数名：`PR_scbc_mc_rc`
   - 默认值：3.27
   - 作用：影响循环的温升、热效率和功率输出

3. **ORC透平膨胀比**：
   - 参数名：`target_PR_orc_expansion_ratio`
   - 默认值：3.37
   - 作用：决定ORC透平的压力降和输出功

4. **ORC透平入口温度**：
   - 参数名：`target_theta_w_orc_turbine_inlet_C`
   - 默认值：127.76°C
   - 作用：影响ORC循环的最高温度和效率

基于这四个关键变量，系统会自动计算其他相关参数：

1. **预冷器出口温度 (T9)**：
   ```
   T9 = base_T9 + sensitivity_factor * (PR - base_PR)
   ```
   - base_T9 = 84.38°C (对应压比3.27)
   - sensitivity_factor = 5.0°C/PR单位

2. **ORC蒸发压力**：
   - 根据ORC透平入口温度和膨胀比计算
   - 默认值：1156.61 kPa

3. **ORC泵入口温度**：
   - 根据冷凝温度和过冷度计算
   - 默认值：49.90°C

## 二、参数详解

以下是对各部分主要参数的解释。请注意，实际文件中的参数名称可能因代码版本或具体实现略有不同，但核心意义相似。

### 1. 通用参数 (`common_parameters`)

这部分参数定义了整个系统运行的环境基准。

*   **`T0_C`**:
    *   **意义**: 环境参考温度，用于㶲分析计算 [5]。
    *   **作用**: 定义了计算㶲值时的"死态"温度。
    *   **单位**: 摄氏度 (°C)
    *   **示例**: `25.0`

*   **`P0_kPa`**:
    *   **意义**: 环境参考压力，用于㶲分析计算 [5]。
    *   **作用**: 定义了计算㶲值时的"死态"压力。
    *   **单位**: 千帕斯卡 (kPa)
    *   **示例**: `101.325`

### 2. SCBC 循环参数 (`scbc_parameters`)

这部分定义了超临界CO2布雷顿循环的各项参数。

*   **`working_fluid_scbc`**:
    *   **意义**: SCBC循环所使用的工质名称 [5]。
    *   **作用**: 供物性计算库 (如 CoolProp) 调用，以获取该工质的热力学性质。
    *   **单位**: 无 (字符串)
    *   **示例**: `"CO2"` 或 `"CarbonDioxide"`

*   **`T1_compressor_inlet_C`**:
    *   **意义**: 主压缩机 (MC) 入口温度 (对应循环中的状态点1) [5]。
    *   **作用**: SCBC循环的初始状态参数之一，显著影响压缩机功耗和循环性能。
    *   **单位**: 摄氏度 (°C)
    *   **示例**: `35.0`

*   **`P1_compressor_inlet_kPa`**:
    *   **意义**: 主压缩机 (MC) 入口压力 (对应循环中的状态点1) [5]。
    *   **作用**: SCBC循环的初始状态参数之一，与压比共同决定循环的最高压力。
    *   **单位**: 千帕斯卡 (kPa)
    *   **示例**: `7400.0` (即 7.4 MPa)

*   **`T5_turbine_inlet_C`**:
    *   **意义**: SCBC透平入口温度 (对应循环中的状态点5) [5]。
    *   **作用**: 关键的循环性能参数，直接影响透平输出功和循环热效率。常作为优化变量 [2][5]。
    *   **单位**: 摄氏度 (°C)
    *   **示例**: `599.85`

*   **`PR_scbc_mc_rc`** (或类似名称如 `target_PR_scbc`):
    *   **意义**: SCBC主循环的总压比 (通常指主压缩机和再压缩机联合作用下的总压比) [5]。
    *   **作用**: 关键的循环性能参数，影响循环的温升、热效率和功率输出。常作为优化变量 [2][5]。
    *   **单位**: 无量纲
    *   **示例**: `3.2`

*   **`split_ratio_rc`**:
    *   **意义**: 再压缩循环中的分流比，即流入再压缩机的CO2质量流量占主压缩机出口总质量流量的比例。
    *   **作用**: 影响回热效果和压缩机功耗，是再压缩S-CO2循环特有的重要参数。
    *   **单位**: 无量纲 (0 到 1之间)
    *   **示例**: `0.35`

*   **`eta_mc`**:
    *   **意义**: SCBC主压缩机的等熵效率 [5]。
    *   **作用**: 描述主压缩机实际压缩过程与理想等熵压缩过程的偏离程度，影响实际功耗。
    *   **单位**: 无量纲 (0 到 1之间)
    *   **示例**: `0.85`

*   **`eta_rc`**:
    *   **意义**: SCBC再压缩机的等熵效率 [5]。
    *   **作用**: 描述再压缩机实际压缩过程与理想等熵压缩过程的偏离程度。
    *   **单位**: 无量纲 (0 到 1之间)
    *   **示例**: `0.85`

*   **`eta_t_scbc`**:
    *   **意义**: SCBC透平的等熵效率 [5]。
    *   **作用**: 描述透平实际膨胀过程与理想等熵膨胀过程的偏离程度，影响实际输出功。
    *   **单位**: 无量纲 (0 到 1之间)
    *   **示例**: `0.90`

*   **`m_dot_scbc_kg_s`** (或在模拟器内部基于其他参数计算):
    *   **意义**: SCBC循环的工质质量流量。
    *   **作用**: 决定系统的整体规模和功率输出。
    *   **单位**: 千克/秒 (kg/s)
    *   **示例**: `1945.09` (主压缩机流量)

### 3. ORC 循环参数 (`orc_parameters`)

这部分定义了有机朗肯循环的各项参数。

*   **`working_fluid_orc`**:
    *   **意义**: ORC循环所使用的工质名称 [5]。
    *   **作用**: 供物性计算库调用，以获取该工质的热力学性质。
    *   **单位**: 无 (字符串)
    *   **示例**: `"R245fa"`

*   **`P_eva_kPa_orc`**:
    *   **意义**: ORC的蒸发压力 [5]。
    *   **作用**: 关键参数，影响ORC的蒸发温度、循环效率和从SCBC乏气中回收的热量。
    *   **单位**: 千帕斯卡 (kPa)
    *   **示例**: `1500.0`

*   **`T_cond_orc_C`** (或 `P_cond_kPa_orc`):
    *   **意义**: ORC的冷凝温度 (或冷凝压力)。
    *   **作用**: 影响ORC的排热量和循环的最低压力/温度，受环境冷却条件制约。
    *   **单位**: 摄氏度 (°C) 或 千帕斯卡 (kPa)
    *   **示例 (`T_cond_orc_C`)**: `35.0`

*   **`target_PR_orc_expansion_ratio`** (或 `PR_orc`) [2][5]:
    *   **意义**: ORC透平的目标膨胀比。
    *   **作用**: 决定ORC透平的压力降和输出功，是ORC性能优化的关键参数。常作为优化变量 [2][5]。
    *   **单位**: 无量纲
    *   **示例**: `3.37`

*   **`target_theta_w_orc_turbine_inlet_C`** (或 `T_orc_turbine_inlet_C` / `new_theta_w_orc_c` [2]):
    *   **意义**: ORC透平入口的目标温度 [5]。
    *   **作用**: ORC循环的最高温度，显著影响ORC效率和输出功。常作为优化变量 [2][5]。
    *   **单位**: 摄氏度 (°C)
    *   **示例**: `127.76`

*   **`delta_T_superheat_orc_K`**:
    *   **意义**: ORC工质在蒸发器出口（即透平入口前）的目标过热度。
    *   **作用**: 确保进入透平的是过热蒸汽，防止液击，并影响循环性能。
    *   **单位**: 开尔文 (K) 或 摄氏度 (°C) (温差)
    *   **示例**: `5.0` (或根据特定工况计算，如 `20.96`)

*   **`delta_T_subcool_orc_K`**:
    *   **意义**: ORC工质在冷凝器出口的目标过冷度。
    *   **作用**: 确保进入工质泵的是液态工质，防止气蚀。
    *   **单位**: 开尔文 (K) 或 摄氏度 (°C) (温差)
    *   **示例**: `2.0`

*   **`eta_t_orc`**:
    *   **意义**: ORC透平的等熵效率 [5]。
    *   **作用**: 描述ORC透平的实际性能。
    *   **单位**: 无量纲 (0 到 1之间)
    *   **示例**: `0.80`

*   **`eta_p_orc`**:
    *   **意义**: ORC工质泵的等熵效率 [5]。
    *   **作用**: 描述ORC工质泵的实际性能。
    *   **单位**: 无量纲 (0 到 1之间)
    *   **示例**: `0.75`

### 4. 换热器参数 (`heat_exchangers_common` 或分散在各循环中)

这部分参数定义了系统中各类换热器的性能。

*   **`eta_L`** (通常在 `scbc_parameters` 内):
    *   **意义**: SCBC低温回热器 (LTR) 的效能 (Effectiveness) [5]。
    *   **作用**: 描述LTR实际换热量与最大可能换热量的比值。
    *   **单位**: 无量纲 (0 到 1之间)
    *   **示例**: `0.86`

*   **`eta_H`** (通常在 `scbc_parameters` 内):
    *   **意义**: SCBC高温回热器 (HTR) 的效能 (Effectiveness) [5]。
    *   **作用**: 描述HTR实际换热量与最大可能换热量的比值。
    *   **单位**: 无量纲 (0 到 1之间)
    *   **示例**: `0.86`

*   **`approach_temp_eva_K_orc`** (或 `min_pinch_eva_orc_K`):
    *   **意义**: ORC蒸发器（即SCBC乏气与ORC工质之间的换热器）的最小接近温差或夹点温差 [5]。
    *   **作用**: 换热器设计的关键约束，确保有效的热量传递。
    *   **单位**: 开尔文 (K) 或 摄氏度 (°C) (温差)
    *   **示例**: `5.0` (或根据假设如 `10.0`)

*   **`approach_temp_cond_K_orc`** (或 `min_pinch_cond_orc_K`):
    *   **意义**: ORC冷凝器的最小接近温差或夹点温差。
    *   **作用**: 换热器设计的关键约束，确保有效的冷量传递。
    *   **单位**: 开尔文 (K) 或 摄氏度 (°C) (温差)
    *   **示例**: `5.0`
